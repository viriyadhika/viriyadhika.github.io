Habit Tracker - Android App

One of the practices that I do to boost my productivity is doing up a Bullet Journal. Inspired by this practice, I designed an app to help me streamline my daily bullet journaling. Habit Tracker is an app designed for tracking a habit, adding a daily record and generating statistics, all within a few clicks.

  • Using Android Studio platform to build
  • Built-using Model-View-ViewModel (MVVM) architecture
  • Utilize SQL-Lite Database via Android Room back-end
  • Front-end is achieved through and Android built-in ViewModel and rendering of XML
  • External library (MPAndroidChart) for displaying statistics
